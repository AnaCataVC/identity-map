## 🇪🇸 Español

### 1. Descripción del Proyecto
Esta es la interfaz gráfica de **IdentityMap**. Construida como una SPA (Single Page Application) moderna, te permite crear, vincular y visualizar los componentes de tu identidad de manera fluida y con físicas de partículas. Toda la información se guarda localmente en tu navegador.

### 2. Tecnologías Utilizadas
- React 19
- TypeScript
- Vite
- TailwindCSS 4
- Framer Motion
- Lucide React

### 3. Instrucciones para Ejecutar en Local

1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Abre tu navegador en `http://localhost:3000`.

## 🇬🇧 English

### 1. Project Description
This is the graphical interface for **IdentityMap**. Built as a modern SPA (Single Page Application), it allows you to fluently create, link, and visualize the components of your identity with particle physics. All data is saved locally in your browser.

### 2. Technologies Used
- React 19
- TypeScript
- Vite
- TailwindCSS 4
- Framer Motion
- Lucide React

### 3. Local Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser to `http://localhost:3000`.
